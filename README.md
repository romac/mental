## Features

- [x] Hindley-Milner type inference
- [x] Let polymorphism
- [x] REPL
- [x] Primitives
- [x] Evaluator
- [ ] Declarations
- [ ] Modules
- [ ] User-defined infix operators
- [ ] Algebraic data types + pattern matching
- [ ] Records + row-polymorphism
- [ ] Type classes
- [ ] Do notation
- [ ] Optimizer / inliner
- [ ] FFI
- [ ] JavaScript / JVM / LLVM backend
- [ ] Integration with SMT solvers (for eg. proof checking)
